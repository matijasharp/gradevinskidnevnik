## Goal
Extract ErrorBoundary, SecretsModal, and AppShell from App.tsx into dedicated files. Restructure AppContent's return to use `<AppShell>` as the layout wrapper, with nav helpers (NavItem, MobileNavItem) co-located in AppShell.

## Purpose
These are the app chrome components — the persistent navigation and layout shell that every view renders inside. They must be extracted before view components (Phases 8–9) can be independently imported, because extracted views will need to be mounted inside a shell.

- `app/src/shared/components/ErrorBoundary.tsx` — class component
- `app/src/shared/components/SecretsModal.tsx` — secrets/API key modal
- `app/src/app/layouts/AppShell.tsx` — sidebar + mobile nav + layout wrapper, includes NavItem + MobileNavItem helpers
- `app/src/App.tsx` — imports added, inline defs removed, AppContent return uses `<AppShell>`
- `npm run build` passes with zero errors

## Decisions Made

| Decision | Rationale | Impact |
|----------|-----------|--------|
| `<img src="/icons/name.svg">` instead of inline SVG | No import churn, simpler markup, SVGs are static assets | Pattern for all future custom icons |
| Sidebar brand text: "Građevinski Dnevnik" (shortened) | Full name too wide for sidebar layout | Tooltip/alt carries full name |
| RootRedirect as separate component | Keeps AppRouter clean; loading check prevents flash | Pattern for auth-aware root routing |
| No GuestRoute wrapper on `/` | RootRedirect handles auth itself; GuestRoute would cause redirect loop | Confirmed: auth→/ → /reports, no loop |

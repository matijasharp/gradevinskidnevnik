<objective>
## Goal
Extract auth state (user, appUser, company, loading, showOnboarding) from AppContent into AuthProvider context, wrap main.tsx with BrowserRouter + AuthProvider, and replace auth useState in AppContent with useAuth().

## Purpose
Decouples auth lifecycle from AppContent so Phase 12 (navigate() migration) can implement auth-aware routing guards in ProtectedRoute/GuestRoute without touching AppContent state.

## Output
- `app/src/app/providers/AuthProvider.tsx` — context, provider, useAuth() hook
- `app/src/app/providers/index.ts` — barrel export
- `app/src/main.tsx` — wrapped with BrowserRouter + AuthProvider
- `app/src/App.tsx` — AppContent auth useState + auth useEffect replaced with useAuth()
</objective>

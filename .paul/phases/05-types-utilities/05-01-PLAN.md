<objective>
## Goal
Create a single source of truth for all shared types and pure utility functions. Replace all duplicate inline definitions in App.tsx and data.ts with imports from the new shared layer.

## Purpose
App.tsx (6,013 lines) and data.ts (1,581 lines) currently duplicate type definitions for Company, AppUser, Project, DiaryEntry, DiaryPhoto and more. This phase establishes the shared foundation that every subsequent extraction phase depends on. Zero behavior change — only file reorganization.

## Output
- `app/src/shared/types/index.ts` — canonical type definitions
- `app/src/shared/utils/format.ts` — safeFormatDate, stripMarkdown
- `app/src/shared/utils/image.ts` — trimCanvas, compressImage
- `app/src/shared/utils/error.ts` — OperationType, FirestoreErrorInfo, setAuthContext, handleFirestoreError
- `app/src/shared/constants/disciplines.ts` — re-export of disciplineConfig
- App.tsx and data.ts updated to import from shared layer
- firebase.ts deleted, firebase npm packages removed
</objective>

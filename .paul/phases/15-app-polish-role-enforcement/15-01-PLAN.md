<objective>
## Goal
Gate master project visibility and creation by org discipline, and create the shared icons folder.

## Purpose
Contractors (non-`general` discipline orgs) must not see the master project nav unless invited, and must not be able to create master projects. Coordinator orgs (`discipline = 'general'`) retain full access. This is required before launch so electrician users don't see a coordinator feature.

## Output
- "Master projekti" nav hidden from contractor orgs with no invited master projects
- "Novi master projekt" create button hidden for contractor orgs
- Contractor empty state shows "no invitations" instead of "create"
- `src/shared/icons/index.ts` barrel export created for custom icon uploads
</objective>

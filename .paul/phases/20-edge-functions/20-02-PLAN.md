<objective>
## Goal
Move PDF generation off the client by creating a Supabase Edge Function using `pdf-lib`. The client calls the Edge Function with diary data and receives a PDF blob for download.

## Purpose
Client-side jsPDF runs in the browser and cannot be called from the Edge Function context. `pdf-lib` is the correct replacement — it is pure JavaScript with no browser dependencies (`window`, `document`, `canvas`) and runs natively in Deno. Image quality is preserved or improved: pdf-lib embeds images from raw bytes directly (no canvas re-encode), whereas jsPDF currently re-encodes photos through canvas at 0.7 JPEG quality.

## Output
- `supabase/functions/generate-pdf/index.ts` — Deno Edge Function, full PDF generation with pdf-lib
- `app/src/integrations/pdf/generateDiaryPdf.ts` — rewritten to call the Edge Function; same exported function signature preserved
</objective>

## Coordinate system note
jsPDF uses top-left origin, mm units, A4 = 210×297mm (content to x=196mm).
pdf-lib uses **bottom-left origin**, **point units** (1mm = 2.835pt), A4 = 595.28×841.89pt.
Use a helper `py(yMm)` to convert: `py(yMm) = 841.89 - (yMm * 2.835)` for drawing from top.
Page overflow condition: jsPDF checks `y > 240` (mm); equivalent in pdf-lib context: track `yMm` and add page when `yMm > 240`.

  <action>
    Create `supabase/functions/generate-pdf/index.ts` as a Deno Edge Function.

    **Imports (top of file):**
    ```
    import { serve } from 'https://deno.land/std@0.168.0/http/server.ts'
    import { PDFDocument, rgb } from 'https://esm.sh/pdf-lib@1.17.1'
    import fontkit from 'https://esm.sh/@pdf-lib/fontkit@1.1.1'
    ```

    **CORS headers** (same pattern as send-invitation):
    ```
    const corsHeaders = {
      'Access-Control-Allow-Origin': '*',
      'Access-Control-Allow-Headers': 'authorization, x-client-info, apikey, content-type',
    }
    ```
    Handle OPTIONS preflight first.

    **Helper functions to define at module level:**

    1. `py(yMm: number, pageHeight = 841.89): number`
       Converts mm-from-top to pdf-lib points-from-bottom: `return pageHeight - (yMm * 2.835)`

    2. `mmToPt(mm: number): number` → `return mm * 2.835`

    3. `hexToRgb(hex: string): [number, number, number]`
       Parse `#RRGGBB` → `[r/255, g/255, b/255]` for use with `rgb()`.
       Default fallback: `[0.227, 0.725, 0.890]` (= #3ab9e3 normalized).

    4. `wrapText(text: string, font: any, fontSize: number, maxWidthPt: number): string[]`
       Word-wraps text: split on spaces, accumulate words, push line when `font.widthOfTextAtSize(line, fontSize) > maxWidthPt`, return array of lines.

    5. `fetchImageBytes(url: string): Promise<Uint8Array | null>`
       Fetch image URL, return `new Uint8Array(await res.arrayBuffer())` on success, `null` on error.

    6. `embedImage(pdfDoc: PDFDocument, bytes: Uint8Array): Promise<any>`
       Try `pdfDoc.embedJpg(bytes)` first; on error try `pdfDoc.embedPng(bytes)`; on both errors return null.

    **PDF generation logic** (inside the POST handler):

    Parse body: `const { project, entries, company, photos } = await req.json()`

    ```
    const pdfDoc = await PDFDocument.create()
    pdfDoc.registerFontkit(fontkit)
    ```

    **Font loading:**
    Fetch Roboto-Regular.ttf from `https://cdnjs.cloudflare.com/ajax/libs/pdfmake/0.1.66/fonts/Roboto/Roboto-Regular.ttf`.
    On success: `const roboto = await pdfDoc.embedFont(fontBytes)`.
    On failure: `const roboto = await pdfDoc.embedFont(StandardFonts.Helvetica)`.

    **Brand color:**
    `const [cr, cg, cb] = hexToRgb(company?.brandColor ?? '#3ab9e3')`
    `const brandColor = rgb(cr, cg, cb)`

    **First page:**
    `let page = pdfDoc.addPage([595.28, 841.89])`
    `let yMm = 15` (working cursor in mm from top)

    **Company logo** (top-right):
    If `company?.logoUrl`: fetchImageBytes → embedImage → if not null:
    `page.drawImage(img, { x: mmToPt(170), y: py(10 + 25), width: mmToPt(25), height: mmToPt(25) })`
    (position mirrors jsPDF: x=170mm, y=10mm from top, 25×25mm)

    **Company name header:**
    `page.drawText(company?.name ?? 'Građevinski Dnevnik Online', { x: mmToPt(14), y: py(22), size: 22, font: roboto, color: brandColor })`

    **Company info lines** (8pt, `rgb(0.59, 0.59, 0.59)`):
    `yMm = 27`
    Draw `company.address` if present, advance `yMm += 4`.
    Draw `${phone} | ${email}` if either present, advance `yMm += 4`.
    Draw `company.website` if present.

    **Project info block** (10pt, `rgb(0.39, 0.39, 0.39)`):
    Fixed positions matching jsPDF: y=42, 47, 52, 57mm:
    - `Izvještaj projekta: {project.projectName}`
    - `Klijent: {project.clientName}`
    - `Adresa: {project.street}, {project.city}`
    - `Datum: {safeFormatDate}` — format date as `dd.MM.yyyy` inline (no import; use toLocaleDateString with Croatian locale or manual format)

    **Divider line** at y=62mm:
    `page.drawLine({ start: { x: mmToPt(14), y: py(62) }, end: { x: mmToPt(196), y: py(62) }, thickness: 0.5, color: brandColor })`

    `yMm = 72`

    **Entry loop** — sort entries newest-first by `entryDate`:
    For each entry:

    Page overflow check: `if (yMm > 240) { page = pdfDoc.addPage([595.28, 841.89]); yMm = 20 }`

    - Entry header (12pt, black): `{safeFormatDate(entry.entryDate)} - {entry.title}`; `yMm += 7`
    - Meta line (9pt, brandColor): `Vrsta posla: {workType} | Status: {status}` + weather/temp if present; `yMm += 6`
    - AI summary if present: wrapText to 180mm width; draw each line (10pt, `rgb(0.24, 0.24, 0.24)`), `yMm += lines.length * 5 + 5`
    - Line items if present:
      - Label `Izvršeni radovi / Stavke:` (9pt); `yMm += 5`
      - Each item: `• {name}: {quantity} {unit}` (9pt); check overflow before each; `yMm += 5`
      - `yMm += 5` after loop
    - Else: `yMm += 5`

    - Photos: filter `photos` by `p.entryId === entry.id`; up to 9:
      - `photoWidth = 55mm → mmToPt(55)`, `photoHeight = 41mm → mmToPt(41)`, `gap = 4mm → mmToPt(4)`
      - 3-column grid layout, same as jsPDF
      - Page overflow: `if (yMm > 230) { page = pdfDoc.addPage(...); yMm = 20 }`
      - Draw label `Fotografije:` (8pt, grey); `yMm += 4`
      - For each photo `pi` (0..min(9, count)-1):
        - `col = pi % 3`, `row = Math.floor(pi / 3)`
        - `xPt = mmToPt(14 + col * 59)`
        - `rowYMm = yMm + row * 45` (41 + 4 gap)
        - Row overflow guard: `if (row > 0 && col === 0 && rowYMm > 245) break`
        - fetchImageBytes(photo.url) → embedImage → drawImage at `{ x: xPt, y: py(rowYMm + 41), width: mmToPt(55), height: mmToPt(41) }`
      - `rows = Math.min(Math.ceil(Math.min(count, 9) / 3), 3)`
      - `yMm += rows * 45 + 4`

    - Signature if `entry.signatureUrl`:
      - Page overflow: `if (yMm > 240) { page = pdfDoc.addPage(...); yMm = 20 }`
      - Label `Potpis odgovorne osobe:` (8pt, grey); `yMm += 2`
      - fetchImageBytes → embedImage → drawImage `{ x: mmToPt(14), y: py(yMm + 15), width: mmToPt(40), height: mmToPt(15) }`
      - `yMm += 18`

    - Entry separator (light grey line): `page.drawLine({ ..., color: rgb(0.94, 0.94, 0.94), thickness: 0.3 })`; `yMm += 5`

    for (let i = 0; i < pageCount; i++) {
      const p = pdfDoc.getPage(i)
      p.drawText(`Stranica ${i+1} od ${pageCount}`, { x: 297.64, y: mmToPt(7), size: 8, font: roboto, color: rgb(0.59,0.59,0.59) })
      p.drawText('Generirano putem Građevinski Dnevnik Online', { x: mmToPt(14), y: mmToPt(7), size: 8, font: roboto, color: rgb(0.59,0.59,0.59) })
    }
    ```

      headers: {
        'Content-Type': 'application/pdf',
        'Content-Disposition': `attachment; filename="${project.projectName}_Izvjestaj.pdf"`,
        ...corsHeaders,
      }
    })
    ```

    **New implementation:**
    ```typescript
    import type { Project, DiaryEntry, Company, DiaryPhoto } from '../../shared/types';
    import { supabase } from '../supabase';

    export const generateDiaryPdf = async (
      project: Project,
      entries: DiaryEntry[],
      company: Company | null,
      photos: DiaryPhoto[] = []
    ): Promise<void> => {
      const { data, error } = await supabase.functions.invoke('generate-pdf', {
        body: { project, entries, company, photos }
      });

      if (error) {
        console.error('PDF generation failed:', error);
        alert('Greška pri generiranju PDF-a. Pokušajte ponovno.');
        return;
      }

      // data is a Blob for binary (application/pdf) responses
      const blob = data instanceof Blob ? data : new Blob([data], { type: 'application/pdf' });
      const url = URL.createObjectURL(blob);
      const a = document.createElement('a');
      a.href = url;
      a.download = `${project.projectName}_Izvjestaj.pdf`;
      document.body.appendChild(a);
      a.click();
      document.body.removeChild(a);
      URL.revokeObjectURL(url);
    };
    ```

    Note: `document.createElement` is still used here — this is fine because generateDiaryPdf.ts runs in the browser. The browser-specific code is now only the download trigger, not the PDF generation itself.

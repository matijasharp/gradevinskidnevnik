<objective>
## Goal
Replace the Express `/api/invite` endpoint with a Supabase Edge Function that sends invitation emails via Resend.

## Purpose
The Express server is a runtime dependency for email delivery. Moving to an Edge Function colocates email sending with Supabase, removes the Express runtime dependency for this feature, and works identically in local dev (supabase functions serve) and production.

## Output
- `supabase/functions/send-invitation/index.ts` — Deno Edge Function calling Resend REST API
- `OrganizationProvider.tsx` updated — `inviteUser()` calls `supabase.functions.invoke()` instead of `fetch('/api/invite')`
</objective>

  <files>supabase/functions/send-invitation/index.ts</files>
  <action>
    Create a Deno Edge Function at `supabase/functions/send-invitation/index.ts`.

    The function must:
    - Handle CORS preflight (OPTIONS) with headers:
      `Access-Control-Allow-Origin: *`
      `Access-Control-Allow-Headers: authorization, x-client-info, apikey, content-type`
    - Accept POST with JSON body: `{ email, name, role, organizationName, inviterName, appUrl }`
    - Read `RESEND_API_KEY` from `Deno.env.get('RESEND_API_KEY')`
    - If key missing: return `Response.json({ sent: false, reason: 'no_api_key' })` — no crash
    - Call Resend REST API via fetch (no SDK — Deno compatible):
      ```
      POST https://api.resend.com/emails
      Authorization: Bearer {RESEND_API_KEY}
      Content-Type: application/json
      body: { from, to, subject, html }
      ```
    - `from`: read `Deno.env.get('RESEND_FROM') ?? 'Gradevinski Dnevnik <noreply@gradevinski-dnevnik.com>'`
    - `subject`: `Pozivnica za tim — {organizationName}`
    - `html`: copy the exact HTML template from `server.ts` lines 147–163 (same styling, same Croatian copy)
      - `roleLabel`: `role === 'admin' ? 'administrator' : 'radnik'`
      - Login button links to `appUrl`
    - On Resend success: return `Response.json({ sent: true })`
    - On Resend error: return `Response.json({ error: 'Failed to send email' }, { status: 500 })`

    Use `serve` from `https://deno.land/std@0.168.0/http/server.ts`.
  </action>

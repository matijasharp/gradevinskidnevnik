<objective>
## Goal
Wire the deployed `generate-pdf` Edge Function as the primary PDF generator, replacing jsPDF client-side generation; fix image aspect ratio distortion in both the Edge Function and the jsPDF fallback.

## Purpose
The Edge Function (pdf-lib + Jost font) produces higher-quality output than client-side jsPDF. Images are currently stretched to a fixed 55×41mm cell regardless of natural aspect ratio — distorting landscape and portrait photos equally. Fixing both paths ensures quality regardless of which runs.

## Output
- `generateDiaryPdf.ts` rewritten to call Edge Function via `supabase.functions.invoke`; jsPDF kept as fallback on invoke error
- Edge Function: photos drawn fit-within 55×41mm bounding box preserving natural aspect ratio
- jsPDF fallback: photos drawn fit-within 55×41mm bounding box preserving natural aspect ratio
</objective>

  <action>
    In the photo loop (around line 238-249), after `embedImage` succeeds and returns `img`, replace the
    fixed-dimension draw call with aspect-ratio-preserving logic:

    1. pdf-lib EmbeddedImage exposes `img.width` and `img.height` (natural pixel dimensions)
    2. Compute fit-within scale:
       ```typescript
       const maxW = mmToPt(55);
       const maxH = mmToPt(41);
       const scale = Math.min(maxW / img.width, maxH / img.height);
       const drawW = img.width * scale;
       const drawH = img.height * scale;
       ```
    3. Compute top-aligned y (images align to top of cell):
       ```typescript
       const yPt = py(rowYMm) - drawH;
       ```
       Explanation: `py(rowYMm)` is the top-of-cell in pdf-lib points. `y` in pdf-lib is bottom-left,
       so bottom = top - height = py(rowYMm) - drawH.
    4. Draw:
       ```typescript
       page.drawImage(img, { x: xPt, y: yPt, width: drawW, height: drawH });
       ```

    Remove the old fixed-dimension draw call:
    ```typescript
    // DELETE this:
    page.drawImage(img, {
      x: xPt,
      y: py(rowYMm + 41),
      width: mmToPt(55),
      height: mmToPt(41),
    })
    ```

    Avoid changing anything outside the photo embed/draw block.
  </action>

<objective>
## Goal
Add a "Preuzmi PDF" button to the diary entry detail modal that generates and downloads a single-entry PDF when clicked.

## Purpose
Users need to download a PDF of an individual diary entry directly from the entry detail view, without navigating to the project reports page which exports the full project.

## Output
- `DiaryEntryDetailModal` gains an optional `onDownloadPdf` async prop and a "Preuzmi PDF" button in the footer action row
- `ProjectDetailPage` wires `onDownloadPdf` to fetch photos for the entry and call `generateDiaryPdf` with the single entry
</objective>

       ```typescript
       export default function DiaryEntryDetailModal({
         entry, project, onClose, onEdit, onAddToCalendar, hasCalendar, company, onDownloadPdf
       }: {
         entry: DiaryEntry, project: Project, onClose: () => void,
         onEdit: (e: DiaryEntry) => void,
         onAddToCalendar: (e: DiaryEntry) => Promise<void>,
         hasCalendar: boolean, company: Company | null,
         onDownloadPdf?: () => Promise<void>
       })
       ```

    3. Add a `pdfLoading` state after the existing `loading` state:
       ```typescript
       const [pdfLoading, setPdfLoading] = useState(false);
       ```

    4. Add a handler after `handleAdd`:
       ```typescript
       const handleDownloadPdf = async () => {
         if (!onDownloadPdf) return;
         setPdfLoading(true);
         try {
           await onDownloadPdf();
         } finally {
           setPdfLoading(false);
         }
       };
       ```

    5. In the footer `<div className="flex gap-3 pt-4 sticky bottom-0 bg-white pb-2">`, insert the
       "Preuzmi PDF" button AFTER the "Uredi" button and BEFORE the "Dodaj u Kalendar" conditional:
       ```tsx
       {onDownloadPdf && (
         <Button
           variant="outline"
           className="flex-1 border-zinc-200"
           onClick={handleDownloadPdf}
           disabled={loading || pdfLoading}
         >
           {pdfLoading ? (
             <Loader2 className="animate-spin mr-2" size={18} />
           ) : (
             <FileDown size={18} className="mr-2" />
           )}
           {pdfLoading ? 'Generiranje...' : 'Preuzmi PDF'}
         </Button>
       )}
       ```

    Avoid changing any other part of the modal.
